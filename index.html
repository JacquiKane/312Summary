Keys

You have created applications for a third party client. Your first application uses HTML/CSS only, your second application uses a UI platform, Bootstrap.

You have learnt how to use a client specification (powerpoint) and translate this into an effective information presentation application.

Search Engine Optimization (SEO) is all about how easily and accurately web pages are accessed by search engines, like Google. Developers need to be aware of design and structure of web pages, so they appear in the SERP (SERP - Search Engine Results Pages).

Appropriate use of semantic tags is critical.

Developers should include alt tag data for images, and use correct information in labels and descriptive fields.

Developers should make sure that information presented is accurate and appropriate.

Key, trending, search terms should be evaluated and integrated into key parts of the web page (like h1, article, main elements), where appropriate.

SEO Analysis engines are available free, and for a fee. These should be used by commercial sites to enhance business outcomes.
