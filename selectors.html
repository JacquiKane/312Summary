Topic: Selectors

When we use the term selector in web development, we mean a rule that targets, or selects, specific elements on a web page, and applies certain styling to these elements.

Selectors are made up of the selector name, and a declaration block. The declaration block is made up of declarations, key-value pairs describing the style.

Selectors can be categorized by the syntax used to embody the style rules.

combinator selectors attribute selectors pseudo-class selectors pseudo-element selectors
like descendant, adjacent, sibling selectors Like attribute only, partial string Like link visited, focus, hover, active Like first-letter

Selectors make it easier to target specific HTML elements.
